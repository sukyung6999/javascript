7장_연산자


  [산술 연산자]

    - 이항 산술 연산자 : 2개의 피연산자가 필요함
     ( + - * / % )

    - 단항 산술 연산자 : 1개의 피연산자가 필요함
      ++ : 피연산자를 바꿈
      -- : 피연사자를 바꿈
      + : 피연산자를 바꾸지 않음
      - : 피연산자를 바꾸지 않음

      예제) var x = false; 
        console.log(-x); // 0
        console.log(x); // false

    - 문자열 결합 연산자: 피연산자 중 하나 이상이 문자일 경우 문자로 반환
      ( + ) 
      ㄴ 암묵적 타입 변환이 이뤄짐


  [할당 연산자]
  ( = )
  우항에 있는 피연산자가 연산이 되어 좌항에 있는 변수에 할당됨
  
  ㄴ 여러 변수에 연쇄적으로 같은 값을 할당할 수 있음
